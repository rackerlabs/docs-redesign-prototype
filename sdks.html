Gophercloud Go

Requirements

  1. Go
  2. Git

Install Gophercloud

Run the following command (after installing Go):

go get github.com/rackspace/gophercloud

Then use it in your code by importing it:

import (
"github.com/rackspace/gophercloud"
"github.com/rackspace/gophercloud/rackspace"
)

JClouds Java

Requirements

  1. Java Development Kit (JDK) version 6 or higher
  2. Maven version 3 or higher

Install JClouds

Download this pom.xml file.

mvn dependency:copy-dependencies "-DoutputDirectory=./lib"

Tip: To make jclouds a part of your existing project, simply copy the relevant dependencies into your own pom.xml file.

pkgcloud Node.js

Requirements

  1. Node.js
  2. NPM

Install pkgcloud

$ npm install pkgcloud
var pkgcloud = require('pkgcloud');

var client = pkgcloud.providers.rackspace.createClient({/* ... */});
